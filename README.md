# Gameoflife

## Project creation

```bash
mix phx.new . --app gameoflife --no-ecto --no-mailer --no-gettext
mix deps.get
mix phx.server
```

## Tailwind

[Tailwind Phoenix guide](https://tailwindcss.com/docs/guides/phoenix)

```bash
mix deps.get
mix tailwind.install
mix phx.server
```

- Step 1: Project creation + Tailwind = 20 minutes.
